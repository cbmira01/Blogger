Angular routes:

-- Unknown routes redirect to /home

/home 
  - nice display of all bloggers
  - each blogger has a link to /list-posts/:bloggerid
  - page has a link to /list-posts

/list-posts
  - display briefly all postings from all bloggers in reverse time order
  - each posting has a link to /read-post/:postid

/list-posts/:bloggerid
  - display briefly all postings from blogger :bid in reverse time order
  - link to /read-post/:postid

/read-post/:postid
  - nice display of posting :postid

/create-blogger
  - create a new blogger
  - redirect to /home

/list-bloggers
  - list all bloggers and allow edit/delete
  - link to /create-blogger
  - link to /delete-blogger/:bloggerid
  - link to /update-blogger/:bloggerid

/delete-blogger/:bloggerid
  - delete blogger :bloggerid
  - redirect to /home

/update-blogger/:bloggerid
  - update blogger :bloggerid
  - redirect to /home

/create-post
  - add a new blog posting
  - redirect to /list-posts/all

/list-posts
  - list all postings and allow edit/delete
  - link to /create-post
  - link to /delete-post/:postid
  - link to /update-post/:postid

/delete-post/:postid
  - delete posting :postid
  - redirect to /home

/update-post/:postid
  - update posting :postid
  - redirect to /home
  
/post-not-found
  - handle case of posting not found
